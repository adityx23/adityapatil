## Highlights

- **Mobile Robotics Autonomy (ROS2 + SLAM + Nav2)**  
  Designed and integrated a full mobile robot autonomy stack using LiDAR, IMU, and encoder feedback, with Cartographer SLAM, Nav2 navigation, and ESP32-based motor control. Validated architecture in Gazebo simulation before hardware integration.

- **Embedded Closed-Loop Control (Discrete-Time Controller on Hardware)**  
  Modeled a thermal system, designed a continuous-time compensator, discretized it using Tustin transform, and deployed the resulting difference-equation controller on an Arduino Mega controlling Peltier modules with PWM.

- **Manipulator Simulation and Kinematic Control (MuJoCo, UR10e)**  
  Implemented and evaluated forward and inverse kinematics control strategies under position and velocity command modes in a physics-based simulation, analyzing motion behavior, convergence, and trajectory quality.
