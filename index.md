## Highlights

- **Embedded feedback control:** Non-PID discrete-time thermostat controller (Tustin-discretized) deployed on Arduino Mega @ 500 ms  
- **Mobile robotics:** ROS2 SLAM + Nav2 autonomy pipeline with sensor fusion (IMU + odometry) and simulation-first validation in Gazebo  
- **IoT hardware:** ESP32 multi-sensor IAQ module with custom 3D-printed mounting (TurtleBot3 Waffle plates printed from STL)
